IDs and Classes DEMO

This is mainBox.

IDs are exclusive. One item can have only One ID and One ID can be assigned to only one Item. Means Each Item can have only one UNIQUE ID.
But Class is not exclusive. Which means a single class can be assigned to multiple Items and a single Item can also have Multiple Classes.

For Example:

  <div id="mainBox" class="redbg">
    This is mainBox.
  </div>

  <div id="mainBox" class="redbg">
    This is second mainBox.
  </div>

      This is not a valid practice.


         But


  <div id="mainBox1" class="redbg">
    This is mainBox.
  </div>

  <div id="mainBox2" class="redbg blackBorder">
    This is second mainBox.
  </div>

     These is a Valid Practice.

**********IDs are generally used to Identify the item in JavaScript, which is why the can not be duplicated as that would be a problem for the algorithm, but the classes are used by the CSS which can be duplicated as Multiple Items can have same style or a Single Item can Have Multiple Styles applied to it.***********
